# Лабораторные работы МАИ

Данный репозиторий содержит коллекцию отчетов по лабораторным и практическим работам, выполненных в процессе обучения в Московском Авиационном Институте. Все документы сверстаны в современной системе подготовки типографских документов Typst, что обеспечивает высокую скорость сборки исходников и строгий внешний вид.

## Особенности оформления

Для оформления всех материалов применяется специализированный автоматизированный шаблон [modern-g7-32](https://github.com/typst-g7-32/modern-g7-32). Данный шаблон позволяет:
* Автоматически формировать содержание, списки иллюстраций и таблиц.
* Поддерживать строгую нумерацию согласно требованиям к оформлению текстовых документов.
* Интегрировать программный код и математические формулы напрямую в текст отчета.

## Структура репозитория

Материалы структурированы по курсам и дисциплинам. В директориях работ можно найти как исходники отчетов (`.typ`), так и скрипты для проведения расчетов (`.py`).

* **2 курс**
  * `computer-science/` — Информатика
  * `devops/` — Практики DevOps
  * `electrotechnics/` — Электротехника
  * `numbering-methods/` — Численные методы
* **3 курс**
  * `basics-of-management-theory/` — Основы теории управления (ТАУ)
  * `bzd/` — Безопасность жизнедеятельности
  * `electrotechnics/` — Продвинутая электротехника
  * `lab-mispis/` — Методы и средства проектирования информационных систем

## Использование и компиляция

Для локальной сборки любого из отчетов потребуется установленный компилятор Typst.

1. Перейдите в директорию нужной лабораторной работы.
2. Выполните команду компиляции:

```bash
typst compile main.typ
```

В результате будет сгенерирован PDF-файл отчета. Исходный код расчетных скриптов на языке Python можно запускать изолированно из директорий `scripts/` или корневых папок лабораторных работ.
